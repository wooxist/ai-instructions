# Example Project

간단한 웹 애플리케이션을 만드는 예시 프로젝트입니다.

## 🎯 프로젝트 목표

사용자 인증과 게시판 기능을 가진 간단한 웹 앱 개발

## 📁 프로젝트 구조

```
example_project/
├── .session/          # 작업 세션 일기
├── .work/            # 작업 관리
│   ├── ROADMAP.md    # 프로젝트 로드맵
│   ├── TODO.md       # 할 일 목록
│   └── tasks/        # 복잡한 작업들
├── .instructions.md  # AI 협업 지침
├── src/              # 소스 코드
└── docs/             # 문서
```

## 🚀 시작하기

### AI와 첫 세션 시작

```
사용자: "세션 시작"
AI: [자동으로 오늘 날짜 세션 파일 생성]
```
