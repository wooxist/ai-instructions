# AI 인스트럭션 설계 가이드북

> AI와의 효과적인 협업을 위한 체계적인 인스트럭션 설계 방법론

## 📖 이 책에 대하여

이 프로젝트는 AI와의 협업을 위한 체계적인 인스트럭션 설계 방법론을 정립하고, 이를 가이드북 형태로 제공하는 것을 목표로 합니다.

- **대상 독자**: 개인 사용자, 소규모 팀, AI 협업 입문자~중급자
- **학습 목표**: 개인 또는 소규모 팀에서 단일 AI 에이전트를 효과적으로 설계하고 활용하는 방법을 배웁니다.
- **주요 내용**: 프롬프트 기초부터 구조화된 인스트럭션 설계, 워크플로우 최적화까지 다룹니다.
