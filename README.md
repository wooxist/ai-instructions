# AI 인스트럭션 설계 가이드북

> AI와의 효과적인 협업을 위한 체계적인 인스트럭션 설계 방법론

## 📖 이 책에 대하여

이 가이드북은 개인 또는 소규모 팀에서 단일 AI 에이전트를 효과적으로 설계하고 활용하는 방법을 다룹니다. 프롬프트 기초부터 체계적인 인스트럭션 설계, 그리고 여러 에이전트를 협업시키는 워크플로우 구축까지, AI를 믿을 수 있는 업무 파트너로 만드는 실용적인 방법론을 제시합니다.

**대상 독자**
- 일반 회사원 (업무 효율 향상)
- 분야 전문가 (AI 접목)
- 호기심 많은 학습자

**독자 특성**
- VS Code 같은 도구 사용 가능
- 코딩 경험 없거나 기초적
- 실용적 결과를 빨리 보고 싶어함

**학습 목표**
- 단일 AI 에이전트를 효과적으로 설계하고 활용하는 방법 습득
- 프롬프트 기초부터 구조화된 인스트럭션 설계까지 체계적 학습
- 실용적이고 바로 활용 가능한 AI 협업 방법론 확립

## 📚 책의 구성

### Part 1: 프롬프트와 인스트럭션의 기초 (1-3장) ✅ 완성

AI와 명확하게 소통하기 위한 기본기를 다집니다. 좋은 질문을 설계하는 방법부터, 재사용 가능한 지시서인 '인스트럭션'을 만드는 4가지 핵심 원칙과 8가지 구성 요소를 배웁니다.

### Part 2: 설계 원칙과 구성 요소 (4-8장) 🚧 작업 중

'문제 → 전략 → 전술'의 흐름에 따라 체계적인 인스트럭션 시스템을 설계합니다.

- **문제 (4장)**: AI 협업 시 발생하는 현실적인 제약들을 이해합니다.
- **전략 (5장)**: 제약을 극복하기 위한 핵심 전략으로 '시스템 설계 원칙'을 배웁니다.
- **전술 (6-8장)**: 전략을 구현하기 위한 구체적인 기술인 '입출력 설계', '에이전트 설계', '워크플로우 설계'를 익힙니다.
