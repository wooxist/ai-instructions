# Workflow Library

재활용 가능한 Workflow들을 보관합니다.

---

## 사용법

### 완성된 Workflow 사용
1. 필요한 Workflow 파일 찾기
2. Story에서 참조 또는 복사해서 수정

### 새 Workflow 만들기
1. `templates/workflow-template.md` 복사
2. 내용 작성
3. 자주 사용하면 이 폴더에 추가

---

## 예시

- `workflow-data-processing.md`: 데이터 수집 → 변환 → 저장
- `workflow-api-development.md`: 설계 → 구현 → 테스트
- `workflow-document-creation.md`: 초안 → 리뷰 → 최종
