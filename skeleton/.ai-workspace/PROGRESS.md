## 💡 사용 가이드

### Task 완료 시
1. 해당 Task 체크박스에 [x] 표시
2. 다음 Task로 "← 현재" 이동
3. 모든 Task 완료 시 Story 체크박스에 [x] 표시
4. Git commit with proper message

### Story 완료 시
1. Story 체크박스에 [x] 표시
2. "작업 히스토리"에 기록
3. 다음 Story로 "← 작업중" 이동
4. ARCHIVE.md로 이동 고려 (Sprint 완료 시)

### Sprint 완료 시
1. 해당 Sprint의 모든 Story를 ARCHIVE.md로 이동
2. 다음 Sprint로 이동
