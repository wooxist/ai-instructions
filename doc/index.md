### 핵심 개념: AI 사고 생태계

2권은 조직의 사고와 실행을 6계층으로 구분하는 **AI 사고 생태계** 모델을 기반으로 합니다:

```
1. 미션 (Mission) - 조직이 존재하는 이유
2. 핵심 가치 (Core Values) - 의사결정 기준
3. 비전 (Vision) - 미래 상태
4. 목표 (Goals) - 전략적 방향
5. 사고 클러스터 (Thinking Clusters) - 사고와 추론 ← 2권 핵심
6. 실행 항목 (Execution Items) - 구체적 행동
```
