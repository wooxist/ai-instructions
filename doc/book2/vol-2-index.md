## 2권의 핵심 개념: AI 사고 생태계

2권은 **AI 사고 생태계 6계층 모델**을 기반으로 조직의 사고와 실행을 구조화합니다:

```
┌─────────────────────────────────────────┐
│  1. 미션 (Mission)                       │  ← 철학적 기반
│     "우리는 왜 존재하는가?"               │
├─────────────────────────────────────────┤
│  2. 핵심 가치 (Core Values)              │  ← 의사결정 기준
│     "우리가 지키는 원칙은 무엇인가?"      │
├─────────────────────────────────────────┤
│  3. 비전 (Vision)                        │  ← 미래 상태
│     "우리가 만들 세상은 어떤 모습인가?"   │
├─────────────────────────────────────────┤
│  4. 목표 (Goals)                         │  ← 전략적 방향
│     "비전을 실현하기 위한 구체적 목표"    │
├─────────────────────────────────────────┤
│  5. 사고 클러스터 (Thinking Clusters)    │  ← 사고와 추론 (2권 핵심)
│     "복잡한 문제를 어떻게 해결할 것인가?" │
├─────────────────────────────────────────┤
│  6. 실행 항목 (Execution Items)          │  ← 구체적 행동
│     "당장 해야 할 작업은 무엇인가?"       │
└─────────────────────────────────────────┘

전략 계층 (상위 3계층): 인간이 주도
실행 계층 (하위 3계층): 인간과 AI 협업 ← 2권 초점
```

**2권의 핵심**: 목표(Goal)를 사고 클러스터(Thinking Cluster)로 분해하고, 인간과 AI가 함께 사고하고 추론하며 실행하는 구조를 설계합니다.

### [15장. 피드백 루프와 지속적 학습](vol-2-part-5-chapter-15.md) **★★★**

**핵심 질문**: 실행 결과를 어떻게 학습으로 전환할까?

* 15.1 피드백 루프의 구조
  * 실행 → 사고 → 전략으로의 피드백
  * 6층 → 5층 → 4층 → 2층 연결
* 15.2 목표 충돌 해결
  * 상충(Conflict): 서로 반대 방향 추구
  * 중복(Overlap): 같은 결과 추구
  * 의존(Dependency): 결과 간 의존성
  * 핵심 가치 기반 해결
