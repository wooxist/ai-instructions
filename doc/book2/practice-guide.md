## 학습 경로

### 경로 1: 입문 (2-6주)

**대상**: AI 사고 체계를 처음 배우는 실무자

**학습 순서**:
```
Week 1-2: Ch 11 기본 실습 (3개)
  → 사고 클러스터 기본 개념
  → 단일 목표 분해
  
Week 3-4: Ch 12 기본 실습 (3개)
  → 복잡한 목표 처리
  → 사고 클러스터 조율
  
Week 5-6: 통합 프로젝트 1 (선택)
  → 간단한 실전 프로젝트
```

**핵심 역량**:
- [ ] 사고 클러스터 설계 가능
- [ ] 단일 목표를 4 Stage로 분해
- [ ] 기본 사고 조율자 역할 이해

**권장 실습**:
1. Ch 11-01: 단순 사고 클러스터 설계
2. Ch 11-02: 4 Stage 사고 프로세스 실행
3. Ch 12-01: 계층적 사고 클러스터 설계
4. Ch 12-02: 메타 조율자 설계

#### 기본 실습

**11-01. 단순 사고 클러스터 설계** ⭐
- **목표**: 간단한 목표를 4 Stage로 분해

- **과제**: 복잡한 사고 흐름을 명확히 표현
- **산출물**: Mermaid 다이어그램 + 설명 문서
- **소요 시간**: 3시간

#### 심화 실습

**11-04. 완전한 사고 클러스터 실행** ⭐⭐⭐

## FAQ

### Q1. 어느 학습 경로를 선택해야 하나요?

**A**: 현재 역할과 목표에 따라 선택하세요:
- **실무자**: 경로 1 (입문) 또는 경로 2 (중급)
- **팀 리더**: 경로 2 (중급) 또는 경로 3 (고급)
- **조직 리더**: 경로 3 (고급) 또는 경로 4 (전문가)

### Q2. 기본 실습만 해도 충분한가요?

**A**: 네, 기본 실습만으로도 핵심 개념을 충분히 습득할 수 있습니다. 심화 실습은 더 깊은 이해와 실전 경험을 원하는 경우에만 선택하세요.

### Q3. 실습을 건너뛰어도 되나요?

**A**: 권장하지 않습니다. 각 실습은 다음 실습의 기초가 됩니다. 특히 Ch 11의 기본 실습은 필수입니다.

### Q4. 팀으로 실습할 수 있나요?

**A**: 네, 특히 Ch 14-15의 실습은 팀 단위가 더 효과적입니다. 역할을 분담하고 결과를 공유하세요.

### Q5. 실습 순서를 바꿔도 되나요?

**A**: 챕터 순서는 지켜주세요 (11→12→13→14→15). 하지만 각 챕터 내의 실습 순서는 유연하게 조정 가능합니다.

### Q6. 실습 결과를 어디에 제출하나요?

**A**: 이 책은 자기주도 학습을 위한 것이므로 제출처가 없습니다. 자신의 GitHub에 공개하거나 팀 내에서 공유하는 것을 권장합니다.

### Q7. 더 많은 예시가 필요해요

**A**: 각 챕터의 본문에 상세한 예시가 있습니다. 또한 통합 프로젝트 섹션의 완전한 사례를 참고하세요.

### Q8. 실습 중 막혔을 때는?

**A**: 
1. 해당 챕터의 본문을 다시 읽기
2. 자가 평가 질문으로 개념 확인
3. 이전 챕터 복습
4. Claude/ChatGPT에게 질문

### Q9. 실습에 필요한 도구는?

**A**:
- 기본: 텍스트 에디터, Markdown 지원
- 중급: Python, Jupyter Notebook
- 고급: Git, 데이터 분석 도구
- 도구는 선택사항이며, 개념 이해가 우선입니다

### Q10. 실습 시간이 부족해요

**A**: 
- 기본 실습만 선택 (18개)
- 자신의 업무와 가장 관련 높은 실습 우선
- 주말/퇴근 후 2-3시간씩 투자
- 팀원과 역할 분담
