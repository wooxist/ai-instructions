#### 실습 1: 복합 목표 분해 연습

**목표**: 복잡한 목표를 3-5개의 독립적인 사고 클러스터로 분해합니다.

**과제**:
1. 자신의 업무나 프로젝트에서 복합 목표 1개를 선택하세요
   - 예: "신제품 출시", "부서 재구성", "연간 마케팅 계획"
2. 이 목표를 3-5개의 사고 클러스터로 분해하세요
3. 각 클러스터에 대해 다음을 정의하세요:
   - 이름 및 담당 영역
   - 주요 사고 목표
   - 필요한 전문성
   - 예상 소요 시간

**평가 기준**:
- 전문성 경계가 명확한가?
- 각 클러스터가 독립적으로 작동 가능한가?
- 모든 클러스터를 합치면 원래 목표를 달성하는가?

**제출 형식**: YAML 파일

**예상 소요 시간**: 2-3시간

**과제**:
1. 실습 1에서 만든 사고 클러스터들 사용
2. 각 클러스터 간의 의존 관계를 파악하세요

#### 실습 3: 목표 충돌 탐지 및 해결

**과제**:
1. 자신의 조직이나 팀의 목표 5-10개를 나열하세요
2. 각 유형의 충돌을 찾아보세요:
   - **상충**: 서로 반대 방향을 추구하는 목표가 있는가?
   - **중복**: 같은 결과를 추구하는 목표가 있는가?
   - **의존**: 다른 목표에 의존하는 목표가 있는가?
3. 각 충돌에 대해:
   - 충돌의 구체적 내용 설명

- 조정된 목표가 실행 가능한가?

**제출 형식**: 충돌 분석 보고서 (Markdown)

#### 실습 4: Python으로 의존성 그래프 구현

**목표**: NetworkX 라이브러리를 활용하여 자동화된 의존성 분석 도구를 만듭니다.

**과제**:
1. Python + NetworkX로 의존성 분석 도구 구현
2. 기능:
   - 사고 클러스터 YAML 파일 읽기
   - 의존성 그래프 생성 및 시각화
   - 위상 정렬로 실행 순서 도출
   - 크리티컬 패스 계산
   - 순환 의존성 탐지 및 경고
3. 실습 1-2의 데이터로 테스트

**평가 기준**:
- 코드가 정상 작동하는가?
- 시각화가 이해하기 쉬운가?
- 순환 의존성을 올바르게 탐지하는가?

**제출 형식**: Python 스크립트 + 실행 결과

**예상 소요 시간**: 4-6시간

**힌트**:
```python
import networkx as nx
import matplotlib.pyplot as plt

# 그래프 생성
G = nx.DiGraph()
G.add_edges_from([('marketing', 'content'), ('marketing', 'design')])

# 위상 정렬
order = list(nx.topological_sort(G))
print(f"실행 순서: {order}")

# 시각화
nx.draw(G, with_labels=True)
plt.show()
```
