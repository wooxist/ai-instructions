## 발견한 문제 / 아이디어

### 💡 파일명 표준화의 가치

**효과**:
1. **명확한 구조**: 파일명만 봐도 권/Part/장 구분 가능
   - `vol-1-part-2-chapter-04.md` → 1권 Part 2의 4장임을 즉시 알 수 있음
2. **확장성 확보**: 3권, 4권 추가 시에도 일관된 패턴 유지 가능
3. **유지보수 용이**: 파일 검색, 정렬이 체계적
4. **자동화 가능**: 스크립트로 파일 처리 시 패턴 활용 가능

**개선점**:
- 기존 파일명(`01-introduction.md`)보다 길지만 정보량 증가
- 알파벳 접미사(`11b`, `11c`) 제거로 일관성 향상

### 💡 링크 업데이트 전략

**효율적 접근**:
1. **계층적 업데이트**: 상위 문서(목차) → 하위 문서(각 장)
2. **패턴 기반 치환**: 반복되는 패턴은 일괄 치환 가능
3. **검증 도구 활용**: 링크 체커로 자동 검증 가능

**시간 소요**:
- vol-1-index.md: 30분 (47개 링크)
- 추정: index.md 30분, practice-guide.md 30분, 각 장 1시간
