# 평가 기준 체크리스트 (Evaluator)

- 명확성(Clarity): 모호어(충분히/적절히 등) 제거, 목적·대상·형식 명시
- 구체성(Specificity): 분량/형식/필수 항목/제약/성공 기준 구체화
- 목표 지향성(Goal-oriented): 목적과 산출물의 정합성, 성공 기준과의 연결
- 일관성(Consistency): 용어·톤·구조 일관, glossary/WRITER 준수
- 구조 충족: 8가지 구성 요소(목적/역할/입력/처리/출력/제약/평가/기초지식)
- 산출물 중심(Output-Driven): 출력 형식·검증 경로 명확, 가능 시 JSON Schema 명시
- Human-in-the-Loop: 중요한 승인/검증 지점 명시
