# 개선안 생성 프롬프트 (Before/After)

목표: 문제 지점을 구체적으로 지적하고, 개선된 문구(After)와 근거를 제시한다.

입력(JSON)
```json
{
  "issue": "모호한 표현: 충분히, 적절히",
  "before": "충분히 자세히 작성해줘",
  "context": "경영진 보고서"
}
```

출력(JSON)
```json
{
  "after": "아래 3가지를 포함하여 500단어 이내로 작성: (1) 요약, (2) 핵심 수치 3개, (3) 리스크 2개.",
  "rationale": "모호어 제거, 형식/분량/성공 기준 구체화"
}
```
