# Story 10: 1권-2권 연결 강화

**담당:** AI Agent  
**예상 시간:** 1일  
**우선순위:** High

---

## 목표

1권과 2권의 논리적 연결성을 강화하고, 독자가 자연스럽게 1권에서 2권으로 전환할 수 있도록 가이드를 작성합니다.

---

## 배경

- Story 09에서 2권 전체 검토 완료
- 2권은 96.3/100의 높은 완성도
- 1권과 2권의 연결 지점 검증 필요
- 학습 경로 일관성 확인 필요

---

## 범위

### In Scope
- 1권 개념 참조 검증
- 2권에서 1권 참조 적절성 확인
- 1권 → 2권 전환 가이드 작성
- 학습 경로 일관성 검증
- 개념 누적 학습 구조 확인

### Out of Scope
- 1권 내용 수정 (1권은 확정됨)
- 새로운 개념 추가
- 링크 검증 (Phase 3에서 처리)

---

## 상세 계획

### Task 01: 1권 개념 목록 정리
**목표**: 1권에서 다룬 핵심 개념 파악

**작업**:
- [ ] 1권 목차 분석
- [ ] 핵심 개념 목록 작성
- [ ] 각 개념의 정의 위치 파악
- [ ] 2권과의 연결 지점 식별

**산출물**: 1권 개념 목록 (concepts-vol1.yaml)

**예상 소요**: 2시간

### Task 02: 2권에서 1권 참조 검증
**목표**: 2권이 1권 개념을 올바르게 참조하는지 확인

**검토 항목**:
- [ ] 1권 개념 언급 적절성
- [ ] "1권에서 배운..." 참조 정확성
- [ ] 개념 재정의 vs 참조 구분
- [ ] 누락된 참조 식별

**산출물**: 참조 검증 리포트

**예상 소요**: 3시간

### Task 03: 학습 경로 일관성 검증
**목표**: 1권 → 2권 학습 흐름이 자연스러운지 확인

**검토 항목**:
- [ ] 난이도 진행 (1권 기초 → 2권 실전)
- [ ] 개념 누적 구조
- [ ] 전제 조건 명시
- [ ] 학습 목표 연결성

**산출물**: 학습 경로 분석 리포트

**예상 소요**: 2시간

### Task 04: 1권 → 2권 전환 가이드 작성
**목표**: 독자를 위한 전환 가이드 작성

**내용**:
- 1권 복습 (핵심 개념 요약)
- 2권 사전 준비 (필요 지식)
- 2권 학습 로드맵
- FAQ (자주 묻는 질문)

**산출물**: transition-guide.md

**예상 소요**: 2시간

### Task 05: 2권 Preface 보강
**목표**: 2권 Preface에 1권 연결 정보 추가

**추가 내용**:
- "1권을 읽은 독자라면..." 섹션
- 1권 핵심 개념 간단 요약
- 2권에서 확장되는 개념
- 1권을 읽지 않은 독자를 위한 가이드

**산출물**: 수정된 Preface

**예상 소요**: 1시간
