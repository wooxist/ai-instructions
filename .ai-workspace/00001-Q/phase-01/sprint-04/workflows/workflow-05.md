# Workflow 05: 실습 가이드 작성

**Story**: Story 06 - 실습 가이드 확장

---

## Task 목록 및 흐름

### Task 01: 현재 실습 현황 파악
**입력**: Ch 11-15 파일들
**출력**: 실습 현황 분석 문서
**다음 조건**: 모든 챕터의 실습 상태 파악 완료

### Task 02: 실습 과제 설계
**입력**: Task 01의 분석 결과 + 각 챕터 학습 목표
**출력**: 챕터별 실습 과제 설계안
**의존**: Task 01 완료
**다음 조건**: 각 챕터당 최소 1개 실습 과제 설계

### Task 03: 실습 과제 작성
**입력**: Task 02의 설계안
**출력**: 각 챕터 파일에 실습 과제 추가
**의존**: Task 02 완료
**다음 조건**: 모든 실습 과제 작성 완료

### Task 04: practice-guide.md 업데이트
**입력**: Task 03의 작성된 실습 과제들
**출력**: 업데이트된 `practice-guide.md`
**의존**: Task 03 완료
**다음 조건**: 모든 실습 과제가 가이드에 포함

---

## 완료 기준

- [ ] Ch 11-15 실습 현황 분석 완료
- [ ] 각 챕터당 최소 1개 실습 과제 설계
- [ ] 모든 실습 과제 작성 완료
- [ ] practice-guide.md 업데이트
- [ ] 실습 과제가 학습 목표와 연결됨
- [ ] 난이도 표시 (기본/심화)
