# Task 01: 2권 파일 분석

**목표:** 2권의 모든 파일을 분석하여 Part, Chapter, Section 구조를 명확히 파악합니다.

---

## 작업 내용

### 1. 파일 목록 확인
`doc/book2/` 디렉토리의 모든 .md 파일을 나열합니다.

### 2. 파일 명명 규칙 분석
- `vol-2-part-X-...` 패턴 분석
- Chapter 번호 체계 확인
- 하위 파일 구조 파악 (00-intro, 01, 02, ...)

### 3. 각 파일의 역할 파악
- Part 도입부 (intro)
- Chapter 본문
- Chapter 하위 섹션

### 4. 파일 맵핑 테이블 작성

---

## 산출물

다음 형식의 마크다운 테이블을 생성합니다:

```markdown
| 파일명 | Part | Chapter | Section | 제목 | 역할 |
|--------|------|---------|---------|------|------|
| vol-2-part-0-preface.md | 0 | - | - | 2권 서문 | 서문 |
| vol-2-part-4-intro.md | 4 | - | - | Part 4 도입 | Part 도입부 |
| vol-2-part-4-chapter-11-00-intro.md | 4 | 11 | 0 | Ch 11 서론 | Chapter 도입 |
| ... | ... | ... | ... | ... | ... |
```

---

## 실행 단계

1. `doc/book2/` 디렉토리 트리 확인
2. 각 파일의 첫 번째 헤더(제목) 추출
3. 파일명과 제목을 매칭하여 테이블 작성
4. Part별로 그룹화하여 구조 시각화

---

## 체크리스트

- [ ] 모든 .md 파일이 테이블에 포함됨
- [ ] Part 구조가 명확히 구분됨 (Part 0, 4, 5)
- [ ] Chapter 번호가 정확함 (Ch 11-16)
- [ ] 분할 파일과 통합 파일이 식별됨
- [ ] 각 파일의 제목이 정확히 추출됨
