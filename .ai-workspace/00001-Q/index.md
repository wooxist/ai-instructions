# 작업 00001-Q: doc/book 디렉토리 분리

**목표:** `doc/book` 디렉토리의 내용을 `doc/book1`과 `doc/book2` 두 개의 새로운 디렉토리로 분리합니다.

**세부 계획:**

1.  `doc/book` 내의 `vol-1-`로 시작하는 모든 파일을 `doc/book1` 디렉토리로 이동합니다.
2.  `doc/book` 내의 `vol-2-`로 시작하는 모든 파일을 `doc/book2` 디렉토리로 이동합니다.
3.  `vol-` 접두사가 없는 나머지 파일들의 처리 방안을 사용자님과 논의합니다.
4.  모든 파일이 이동된 후, 원본 `doc/book` 디렉토리를 삭제합니다.
