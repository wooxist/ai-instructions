## 진행 방식

### Story당 프로세스

```
1. 인간 검토 (독자 시점)
   - 내용 이해도
   - 표현 명확성
   - 예제 적절성
   - 흐름 자연스러움

2. 개선 요청 (발견 시)
   - 문제 설명
   - 개선 방향 제시

3. AI 개선 계획
   - 문제 분석
   - 해결 방안
   - 영향 범위

4. 인간 승인

5. AI 실행
   - 파일 수정
   - 문서화

6. 인간 확인

7. Story 완료 표시
```

**진행 중 Story**: `PROGRESS.md`에 표시
