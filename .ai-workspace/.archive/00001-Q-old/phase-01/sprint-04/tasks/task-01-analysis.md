### Ch 15: 피드백 루프와 지속적 학습 (⭐⭐⭐⭐ 심화)

**실습 현황**: ✅ **부분적으로 있음** (체크리스트만)

**현재 실습 구성**:

1. **이해도 체크리스트**
   - [ ] 피드백 루프 3단계 설명 가능
   - [ ] AI와 인간의 역할 분담 이해
   - [ ] 조정의 3가지 수준 구분
   - [ ] 목표 충돌 탐지 및 해결
   - [ ] Blameless Postmortem 작성
   - [ ] PDCA 사이클 적용

2. **기본 실습 과제 (간략)**
   1. 결과 수집 체계 설계
   2. 데이터 분석 및 인사이트 도출
   3. 조정 계획 수립
   4. 목표 충돌 분석
   5. Blameless Postmortem 작성
   6. PDCA 사이클 계획 및 실행

**개선 필요**:

#### 추가 필요 기본 실습 (상세 가이드)
1. **피드백 루프 1단계: 결과 수집**
   - 현재 운영 중인 사고 클러스터 선택
   - 정량 지표 정의 (예: 완료율, 처리 시간, 비용)
   - 정성 피드백 채널 구축 (예: 설문, 회고 미팅)
   - 수집 도구 및 주기 결정
   - 실제로 2주간 데이터 수집

2. **피드백 루프 2단계: 패턴 분석**
   - AI 도구로 통계적 패턴 탐지 (상관관계, 추세, 이상치)
   - 인간이 비즈니스 맥락 해석 (인과관계, 영향 평가)
   - 최소 3가지 인사이트 도출
   - 인사이트 보고서 작성 (템플릿 제공)

3. **피드백 루프 3단계: 조정**
   - Level 1 (실행 항목): 프롬프트, 파라미터 개선
   - Level 2 (사고 클러스터): 사고 단계 추가/제거
   - Level 3 (목표): 목표 수정 또는 추가
   - 조정 계획 수립 및 승인 프로세스

4. **목표 충돌 해결 실습**
   - 조직의 모든 목표 나열 (최소 5개)
   - AI 도구로 상충/중복/의존 관계 분석
   - 각 유형별 해결 전략 수립
   - 우선순위 결정 (핵심 가치 기반)

#### 추가 필요 심화 실습 (프로젝트 기반)
5. **Blameless Postmortem 프로젝트**
   - 최근 실패 사례 1개 선택
   - 타임라인 작성 (무슨 일이 일어났나)
   - 근본 원인 분석 (5 Whys 기법)
   - 학습 사항 정리 (기술적, 프로세스, 조직적)
   - 예방 조치 계획 (Action Items)
   - 팀과 공유

6. **PDCA 사이클 완전 실행**
   - Plan: 개선 목표 및 가설 수립 (1주)
   - Do: 파일럿 실행 (2주)
   - Check: 결과 분석 및 가설 검증 (1주)
   - Act: 전면 적용 또는 개선 (1주)
   - 전체 과정을 문서화하고 회고

7. **학습 문화 구축 프로젝트**
   - 팀 내 피드백 문화 진단 (현재 수준)
   - 심리적 안전감 확보 방안 제시
   - 정기 회고 프로세스 설계 (주간, 월간)
   - 실패 사례 데이터베이스 구축
   - 3개월 후 효과 측정
