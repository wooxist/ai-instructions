## 검증 방법

### 자동 검증 (가능한 경우)
```bash
# 상대 경로로 링크된 파일 존재 확인
cd doc/book2
for link in $(grep -oP '\[.*?\]\(\K[^)]+' vol-2-index.md); do
  if [[ $link != http* ]]; then
    if [ ! -f "$link" ]; then
      echo "❌ 깨진 링크: $link"
    else
      echo "✅ 정상: $link"
    fi
  fi
done
```

### 수동 검증
1. VS Code에서 각 링크를 Cmd+Click하여 이동 확인
2. 이동한 파일에서 해당 섹션(앵커) 존재 확인
