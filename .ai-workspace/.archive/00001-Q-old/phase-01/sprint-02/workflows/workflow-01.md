# Workflow 01: 2권 목차 생성 및 통합

**Story**: Story 02 - 2권 통합 목차 생성 및 메인 목차 업데이트

---

## Task 목록 및 흐름

### Task 01: 2권 파일 분석
**입력**: `doc/book2/` 디렉토리
**출력**: 파일 맵핑 테이블
**다음 조건**: 모든 파일이 테이블에 포함

### Task 02: vol-2-index.md 생성
**입력**: Task 01의 테이블 + Part 도입부 파일들
**출력**: `doc/book2/vol-2-index.md`
**의존**: Task 01 완료 필요
**다음 조건**: 파일 생성 + AI 사고 생태계 다이어그램 포함

### Task 03: 메인 목차 업데이트
**입력**: Task 02의 vol-2-index.md + 기존 `doc/index.md`
**출력**: 업데이트된 `doc/index.md`
**의존**: Task 02 완료 필요
**다음 조건**: 2권 섹션 추가됨

### Task 04: 검증 및 정리
**입력**: Task 02, 03의 결과물
**출력**: 검증 완료 체크리스트
**의존**: Task 02, 03 완료 필요
**다음 조건**: 모든 링크 정상 작동
