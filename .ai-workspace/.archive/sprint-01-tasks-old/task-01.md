# Task 01: 인간 검토

**목표**: 독자 시점에서 챕터 검토

**실행 방법**: 
대상 챕터를 독자 입장에서 읽고 다음 관점으로 평가
- 내용 이해도: 설명이 명확한가?
- 표현 적절성: 용어와 문체가 적절한가?
- 예제 품질: 예제가 이해를 돕는가?
- 흐름 자연스러움: 문단 간 연결이 매끄러운가?

**사용 도구**: 
- 마크다운 뷰어 또는 에디터
- 체크리스트

**체크리스트**:
- [ ] 챕터 전체 읽기 완료
- [ ] 개선 필요 사항 기록
- [ ] 개선 불필요 시 완료 표시

**산출물**: 
- 개선 필요: 피드백 문서 (문제점 + 개선 제안)
- 개선 불필요: "검토 완료, 개선 불필요" 확인
