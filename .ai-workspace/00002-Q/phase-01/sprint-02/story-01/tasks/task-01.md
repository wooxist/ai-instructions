# Task 1: 목차 및 구조 개편

## 목적
Chapter 5의 전체 구조를 '제약 조건 → 해결 방안' 형태로 변경하고, 섹션 제목을 수정하여 내용의 효용성을 높인다.

## 입력
- `doc/book1/vol-1-part-2-chapter-05.md`
- `doc/book1/vol-1-index.md`

## 출력
- 구조와 제목이 수정된 `vol-1-part-2-chapter-05.md` 파일
- 변경된 목차가 반영된 `vol-1-index.md` 파일

## 실행 방법
1.  `vol-1-part-2-chapter-05.md` 파일 열기
2.  H1 제목을 "AI 협업의 4가지 제약 조건과 해결 원칙"과 같이 긍정적이고 실용적인 내용으로 변경
3.  '보안 위험' 관련 섹션 완전 삭제
4.  나머지 4가지 항목(환각, 실시간 정보, 기억력, 상식)의 제목을 '제약 조건'과 '해결 방안'이 드러나도록 수정
    - 예: "환각(Hallucination)" → "제약 1: 사실과 다른 정보 생성 가능성"
5.  `vol-1-index.md` 파일을 열고, Chapter 5의 하위 목차를 위 변경사항에 맞게 정확히 수정

## 완료 기준
- `vol-1-part-2-chapter-05.md`의 목차 구조가 '문제-해결' 중심으로 변경됨
- '보안 위험' 항목이 완전히 제거됨
- `vol-1-index.md`의 Chapter 5 목차가 최신 상태로 반영됨
