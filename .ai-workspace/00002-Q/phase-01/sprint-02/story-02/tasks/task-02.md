# Task 2: 10장 파일명 변경 및 내용 수정 (10장 → 8장)

## 목적
기존 10장(실전 패턴)을 8장으로 재넘버링하고, Part 4에서 Part 3으로 이동한다.

## 입력
- `doc/book1/vol-1-part-4-chapter-10.md` (실전 패턴: 단일 에이전트 설계)

## 출력
- `doc/book1/vol-1-part-3-chapter-08.md` (새로운 8장)

## 실행 방법

### 1단계: 파일 복사
1. `doc/book1/vol-1-part-4-chapter-10.md` 읽기
2. 전체 내용을 복사하여 새 파일에 작성 준비

### 2단계: 내용 수정
파일 내 모든 "10장" 관련 표현을 "8장"으로 변경:

**변경 대상**:
1. H1 제목: `# 10장. 실전 패턴` → `# 8장. 실전 패턴`
2. 모든 섹션 번호: 
   - `## 10.1` → `## 8.1`
   - `### 10.1.1` → `### 8.1.1`
   - `## 10.2` → `## 8.2`
   - (모든 10.X 형태를 8.X로 변경)
3. 본문 내 "10장" 텍스트 → "8장"
4. 앵커 링크: `#101` → `#81`, `#102` → `#82` 등
5. Part 관련 표현: "Part 4" → "Part 3" (있는 경우)

### 3단계: 새 파일 작성
수정된 내용을 `doc/book1/vol-1-part-3-chapter-08.md`에 작성

## 완료 기준
- [ ] vol-1-part-3-chapter-08.md 파일 생성 완료
- [ ] 모든 "10장" 텍스트가 "8장"으로 변경됨
- [ ] 모든 섹션 번호(10.X)가 8.X로 변경됨
- [ ] 모든 앵커 링크가 올바르게 수정됨
- [ ] Part 표현이 수정됨 (있는 경우)
- [ ] 내용의 의미는 변경되지 않음

## 주의사항
- 기존 vol-1-part-4-chapter-10.md는 이 Task에서 삭제하지 않음 (Task 4에서 처리)
- 번호와 Part만 변경하고, 실제 내용은 수정하지 않음
- 변경 시 정규표현식이나 일괄 치환 사용을 고려하여 누락 방지
