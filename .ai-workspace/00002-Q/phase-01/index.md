# Phase 1: 프로그래밍 코드 제거

## 목표
Book1과 Book2의 모든 마크다운 파일에서 일반 독자가 이해하기 어려운 프로그래밍 언어 코드를 제거합니다.

## 배경
- 책의 주요 독자는 코딩 경험이 없거나 기초적인 일반 사용자
- 프로그래밍 코드는 독자의 이해를 방해하고 책의 접근성을 낮춤
- YAML, JSON 같은 설정 파일 형식은 개념 설명에 유용하므로 유지

- Book2: 40개 파일
