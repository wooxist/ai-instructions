# 2장. 인스트럭션 라이프 사이클

---

## 🎯 목적
- 라이프사이클 단계와 산출물을 이해한다.  
- 각 단계의 검증·평가 기준을 정의한다.  
- 반복 가능한 개선 루프를 수립한다.  

---

## 📌 요약
인스트럭션은 일회성 프롬프트가 아니라, 목적 설정부터 개선·축적까지 이어지는 순환 과정입니다. 이 장은 전체 여정을 한눈에 보여주고, 각 단계에서 무엇을 산출해야 하는지 요약합니다.
